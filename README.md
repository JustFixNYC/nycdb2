# nycdb2

Definition repository for NYC housing datasets consumed by [opendata-etl](https://github.com/JustFixNYC/opendata-etl): YAML datasets, dbt models, read-only API specs, and docs.

See [docs/index.md](docs/index.md) for layout, validation commands, and a sample `definitions.yml` snippet using a local `file://` clone URL.

Column naming (`source_header`, `source_skip`, new-column checks): [opendata-etl `docs/column-names.md`](https://github.com/JustFixNYC/opendata-etl/blob/main/docs/column-names.md).

Legacy nycdb migration drafts: [docs/legacy-nycdb-importer.md](docs/legacy-nycdb-importer.md).
